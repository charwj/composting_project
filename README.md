# composting_project
Data for my Medium article on NYC composting programs.

The following files are available in this folder:
- `Final_Project.R` and `Final_Project.Rproj` contain the R code where I analyzed composting data
- `DSNY_Monthly_Tonnage_Data_20251130.csv` contains the DSNY-collected [monthly tonnage data](https://data.cityofnewyork.us/City-Government/DSNY-Monthly-Tonnage-Data/ebb7-mvp5/about_data)
- `Dem_19-23_CDTA.csv` contains the 2023 NYC community district [population estimates](https://www.nyc.gov/content/planning/pages/resources/datasets/american-community-survey)
- The below files were created by scraping data from analyses conducted by Stephanie MacBride:
  - `FY24_CD_capture`
    - Data scraped from tables on pages 44 - 45 of this [CY 2024 Quarterly Capture Rate Report](https://www.academia.edu/127655557/New_York_City_Residential_Curbside_Organics_Program_Fall_2024_CY_2024_Quarterly_Capture_Rate_Report)
  - `FY25_CD_capture`
    - Data scraped from table on page 37 of this [Spring 2025 Quarterly Capture Rate Report](https://www.academia.edu/143278620/New_York_City_Residential_Curbside_Organics_Program_Spring_2025_Quarterly_Capture_Rate_Report)
  - `FY25_boro_capture`
    - Data scraped from table on page 5 of this [Spring 2025 Quarterly Capture Rate Report](https://www.academia.edu/143278620/New_York_City_Residential_Curbside_Organics_Program_Spring_2025_Quarterly_Capture_Rate_Report)
